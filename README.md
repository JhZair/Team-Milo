Team Milo: Director de Marketing de Bolsillo

Aplicación móvil con IA que automatiza la creación de contenido y el cierre de ventas para Mipymes en Perú. Nuestra solución ataca la "Invisibilidad Digital" que causa que 9 de cada 10 microempresas quiebren antes del primer año.



Tecnologías Principales
Backend: Flask (Python) con conexión a MySQL. * Frontend: React + Vite (Web App Móvil). * IA: Motor de tendencias locales y generación de guiones proactivos.



Modelo de Negocio
1. SaaS: Suscripción mensual para acceso a la IA. 2. Ad-Wallet: Billetera prepago para anuncios en Meta Ads, recargable vía Yape o Plin.


Instalación del Backend

    cd backend
    python3 -m venv venv
    source venv/bin/activate
    pip install -r requirements.txt
